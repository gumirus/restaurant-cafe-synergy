## 🛠 Как сделать скриншоты кода

### Способ 1: Вручную (рекомендуется)
1. Открыть файл в VS Code
2. Выделить нужный участок кода
3. Cmd+Shift+P → "Print Screen" или просто Cmd+Shift+4

### Способ 2: Через скрипт
```bash
cd docs
# Установить highlight: brew install highlight
bash generate_screenshots.sh
# Открыть .html файлы в браузере и сделать скриншоты
```

### Способ 3: Carbon (красиво)
1. Зайти на https://carbon.now.sh
2. Вставить код
3. Скачать PNG
