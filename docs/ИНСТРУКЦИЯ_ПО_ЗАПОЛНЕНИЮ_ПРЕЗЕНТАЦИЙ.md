## 🛠 Как сделать скриншоты кода

### Способ 1: Вручную (рекомендуется)
1. Открыть файл в VS Code
2. Выделить нужный участок кода
3. `Cmd+Shift+P` → "Print Screen" или просто `Cmd+Shift+4`

### Способ 2: Через скрипт
```bash
cd docs
# Установить highlight: brew install highlight
bash generate_screenshots.sh
# Открыть .html файлы в браузере и сделать скриншоты
```

### Способ 3: Carbon (красиво)
1. Зайти на https://carbon.now.sh
2. Вставить код
3. Настроить тему (Dracula, One Dark и т.д.)
4. Скачать PNG

### Способ 4: VS Code Extension
1. Установить расширение "Polacode" или "CodeSnap"
2. Выделить код
3. `Cmd+Shift+P` → "Polacode: Screenshot"

## 📸 Рекомендации по скриншотам

### Общие правила
- **Размер:** 1920×1080 или пропорционально
- **Формат:** PNG (без потери качества)
- **Фон:** Белый или светлый (если не Carbon)
- **Без лишних элементов:** обрезать панели инструментов, вкладки

### Для кода
- Использовать тёмную тему (Dracula, One Dark Pro)
- Шрифт: Fira Code, JetBrains Mono или Cascadia Code
- Размер шрифта: 14-16px

### Для страниц
- Полностраничный скриншот (Full Page Screenshot)
- Без полос прокрутки
- Единый масштаб (100%)
